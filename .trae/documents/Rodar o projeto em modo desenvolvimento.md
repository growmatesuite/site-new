## Objetivo
Iniciar o servidor de desenvolvimento (Vite) para acessar o projeto em `http://localhost:5173` sem encerrar o preview já em execução.

## Passos
1. Usar um novo terminal para não interromper o preview atual.
2. Executar: `npm run dev -- --open` na pasta do projeto.
3. Confirmar que o servidor subiu e capturar a URL (padrão `http://localhost:5173`). Se a porta estiver ocupada, iniciar em `--port 3000`.
4. Entregar o link clicável para você acessar.

## Observações
- Não haverá alterações de código; apenas execução de comandos.
- O preview existente em `http://localhost:4173` permanece ativo.
